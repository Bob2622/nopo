NoPo
====

NoPo(no-person-operation) is task-based build / process tool for web project (html/js/css) based on NodeJS, like jake / grunt / ant.

default cmds:
qzmin
concat
minify
beautify
exec
package
autosprite
comment
svnverize
tmplextract
translate
filesync
npmpublish
jshint
jslint



modules
file-utils (copy, copyR, readdirR, rmRf)
file-list
task
event
logging
output

core
